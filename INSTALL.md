=======
Install
=======

basicaly run python setup.py install.

Urpmi
-----

urpmi mandriva-control-center

Requirements
============

 * python-dbus 0.83.2

Backend
-------

 * systemd 17
 * libuser 0.57.1
 * libuser-python 0.57.1

Frontend
--------

 * qt4 4.7.1
 * pyside 1.0.0-0.beta5
 
Documentation
=============
 
Run ``make html`` in docs folder to generate mandriva control center
developer documentation.
